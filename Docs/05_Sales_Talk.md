# 営業トーク: SkyMenu Insight 導入提案シナリオ

## 1. ターゲット：あぜみ社長（多忙な経営層）へのアプローチ

### 【フック：共感と課題の提示】
「あぜみ社長、今日もお忙しい中お疲れ様です。三田のオフィスを出る際、『今日の夕飯、何にしようかな』と考えることすら、少し重荷に感じることはありませんか？ 脳が最も疲れている夕方に、さらに意思決定を強いるのは、経営リソースの浪費かもしれません。」

### 【ソリューションの提示】
「私たちが提案するのは、単なるレシピアプリではありません。あぜみ社長の専属コンシェルジュとして、三田の空模様と社長のコンディションを読み解き、今夜最高のひと皿を提案する『SkyMenu Insight』です。」

### 【価値の強調：データと根拠】
「IISEでソートリーダーシップを牽引される社長なら、データに基づいた最適解の価値をご理解いただけるはずです。例えば、今日の三田は湿度が80%を超えています。これは身体に『重だるさ』を感じさせるサインです。AIは、あえて火を通した温かい豆料理を提案します。これで明日の朝のパフォーマンスが劇的に変わります。」

## 2. ターゲット：IISE人事・総務部門（福利厚生）へのアプローチ

### 【課題の提示】
「知的生産性が最大の資産であるシンクタンクにおいて、従業員のコンディション管理は重要な経営課題です。しかし、食生活の指導はプライバシーもあり、なかなか踏み込めない領域ではないでしょうか？」

### 【ソリューションの提示】
「『SkyMenu Insight』を福利厚生として導入いただくことで、従業員は『今日の天気』という自然なきっかけで、自身の健康をケアする習慣を身につけます。三田という素晴らしい環境を、食を通じて最大限に享受する。そんな新しい働き方を提案しませんか？」

## 3. 想定される反論への対応 (Q&A)

- **Q: 既存のレシピアプリで十分では？**
  - **A**: 「既存アプリは『あるもの』から考えますが、本サービスは『あるべき姿（コンディション）』から考えます。意思決定の質が根本的に異なります。」
- **Q: 三田以外でも使えるのか？**
  - **A**: 「もちろんです。しかし、三田という特定のエリアに特化したローカル情報を深く掘り下げることで、他のアプリにはない『手触り感のある提案』を実現しています。」

## 4. クロージング
「あぜみ社長、まずは1週間、三田の空とAIが選ぶ夕食を試してみませんか？ 自然のバイオリズムに身を委ねることで、新しいインスピレーションが湧いてくるはずです。」
