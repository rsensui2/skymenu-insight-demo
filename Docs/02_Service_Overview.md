# サービス概要: SkyMenu Insight

## 1. サービス名称
**SkyMenu Insight（スカイメニュー・インサイト）**
〜天気が教える、今日のあなたに最高のひと皿〜

## 2. サービスの目的
気象の変化は、私たちの食欲、代謝、そして体調に密接に関係しています。「SkyMenu Insight」は、リアルタイムの気象データ（気温、湿度、気圧、日照時間）を解析し、ユーザーの現在のコンディションに最も適した夕食の献立をパーソナライズして提供することを目的とします。

## 3. 主要機能
- **ウェザー・アナリティクス・エンジン**: 
  現在地（三田エリア等）の気象予測に基づき、身体が欲する栄養素を算出。
  - 例：急激な気温低下 → 代謝を高める根菜やスパイス。
  - 例：低気圧の接近 → むくみを解消するカリウム豊富な食材。
- **スマート献立提案**:
  「自炊（時短レシピ）」「デリバリー（三田周辺）」「外食（高級店）」の3つのモードで提案。
- **バイタル連携（オプション）**:
  ウェアラブルデバイスと連携し、睡眠の質やストレスレベルを加味したさらなる最適化。

## 4. ユーザー体験（UX）
1. **16:00のプッシュ通知**: 「今日の三田は夜から冷え込みます。体温を上げる『生姜たっぷりの豚汁』はいかがですか？」
2. **ワンタップ・アクション**: 提案された献立に必要な食材をネットスーパーのカートに追加、または周辺のレストランを予約。
3. **フィードバック**: 食後の満足度を記録し、AIの学習精度を向上。

## 5. ターゲットユーザー像
- **IISE等のシンクタンク勤務者**: 常に情報を扱い、論理的な根拠を好む層。
- **三田エリアの居住・勤務者**: 生活の質を重視し、効率的なソリューションを求める層。
- **あぜみ社長のようなリーダー層**: 自身の健康管理を「経営課題」として捉えている層。

## 6. 独自性
既存のレシピアプリが「冷蔵庫の余り物」から考えるのに対し、本サービスは「外部環境（天気）」から逆算して「身体の最適解」を導き出す点が最大の特徴です。
